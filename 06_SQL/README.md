# SQL — Data checks (TaskFlow)

В этом разделе показаны типовые SQL-проверки, которые выполняет QA:
- целостность данных (orphans)
- дубликаты
- валидация статусов/приоритетов
- отчётные запросы для проверки корректности бизнес-логики

Файлы:
- `schema.sql` — упрощённая схема БД продукта
- `sql_checks.sql` — набор QA-проверок
