# Regression Checklist — TaskFlow

## Auth & Sessions
- [ ] Сессия сохраняется после обновления страницы
- [ ] Без авторизации нельзя открыть доску по прямой ссылке
- [ ] Истечение сессии/логаут корректно возвращает на логин

## Data Integrity
- [ ] После создания/редактирования данные сохраняются и видны после refresh
- [ ] Удалённые доски/колонки/карточки не появляются снова
- [ ] Нет дублей при повторной отправке форм (double click / slow network)

## Validation
- [ ] Пустые обязательные поля → ошибка
- [ ] Длина названия/описания → корректные ограничения
- [ ] Email формата invalid → ошибка

## Cards workflow
- [ ] Перемещение не ломает порядок карточек
- [ ] Карточки остаются в правильных колонках после refresh
- [ ] Редактирование не сбрасывает статусы/поля

## Cross-browser (basic)
- [ ] Chrome: ключевые сценарии проходят
- [ ] Firefox: ключевые сценарии проходят
